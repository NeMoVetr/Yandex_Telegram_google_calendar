# Инструкция по запуску парсера

## Главный файл
Главный файл запуска парсера: **`google_calendar.py`**

## Подготовка к запуску
Перед запуском необходимо создать следующие API-ключи:
- Telegram
- Яндекс Почты
- Google Календаря
